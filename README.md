# Chức năng
- Tạo ví Solona hàng loạt

# Cách dùng

B1. Tải mã nguồn
```
git clone https://github.com/mhqb365/create-solona-wallet.git
```

B2. Cài đặt
```
cd create-solona-wallet
npm i
```

B3. Tạo ví
```
node create x
```
x: số lượng ví cần tạo, ví dụ tạo 100 ví solona:
```
node create 100
```

Ví sẽ được lưu dạng JSON trong folder wallets
